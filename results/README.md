# Results

This directory may contain aggregate metrics, evaluation summaries, and publication figures. Row-level predictions, model binaries, and machine-specific inventories are ignored by Git.

Important tracked outputs include:

```text
results/prl/evaluation/evaluation_metrics.csv
results/prl/evaluation/evaluation_summary.md
results/prl/evaluation/*.png
results/prl/paper/paper_results_draft.md
results/prl/paper/*.csv
results/prl/paper/*.png
```

All results must be regenerated from the documented pipeline before a tagged release.

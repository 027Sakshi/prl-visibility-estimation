# Generated models

Serialized model files are generated locally and are not committed to Git.

The selected PRL inference bundle is produced at:

```text
models/prl/prl_visibility_model.joblib
```

Generate it with:

```powershell
python run_prl_pipeline.py
```

For a formal release, publish versioned model artifacts separately through a GitHub Release or an archival repository, together with checksums, training configuration, dependency versions, and data provenance.
